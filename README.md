# AstrBot 英文过滤插件

拦截 AstrBot 输出（Bot 回复）中的英文字母，如有则清理后输出，无英文则原样输出。
**清理失败时不做处理，直接发送原消息。**

## 功能

- **仅拦截 Bot 的回复消息**，不影响用户输入
- 自动检测 Bot 回复中的英文字母（a-z, A-Z）
- 包含英文时：移除所有英文字母，保留中文、数字、标点等
- 不包含英文时：原样输出，不做任何修改
- **清理失败时：不做处理，原消息正常发送（防丢消息）**

## 安装

1. 将本插件文件夹复制到 AstrBot 的插件目录：
   ```
   cp -r astrbot_plugin_filter_english /path/to/astrbot/data/plugins/
   ```

2. 重启 AstrBot，插件会自动加载

## 配置

在 AstrBot 管理面板中配置以下选项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `remove_spaces` | bool | true | 清理英文后是否移除多余空格 |
| `log_filtered` | bool | false | 是否在日志中记录过滤操作 |

## 效果示例（Bot 回复）

| Bot 原文 | 过滤后输出 |
|----------|-----------|
| `Hello 你好 world` | `你好` |
| `这是一个test消息` | `这是一个消息` |
| `纯中文内容` | `纯中文内容`（不变） |
| `123ABC456` | `123456` |
| `Hello, World! 你好!` | `, ! 你好!` |

## 失败处理

如果过滤过程中发生任何异常（如正则编译错误、内存不足等），
插件会记录错误日志，但**不会阻止原消息发送**，确保消息不会丢失。
